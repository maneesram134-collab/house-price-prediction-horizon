# House Price Prediction using Machine Learning

## Project Overview
This project builds a machine learning model to predict house prices based on various features.

## Key Results
- **Best Model**: Random Forest
- **R² Score**: 86% accuracy
- **RMSE**: $38,195
- **Mean Error**: 6.04%

## Features Used
- Size (sq ft)
- Bedrooms & Bathrooms
- Age of house
- Location Quality
- Distance to city center
- Neighborhood type
- Garage spaces
